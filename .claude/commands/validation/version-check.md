## Required Versions:
1. Shopify API version ≥ 2024-01
2. Next.js ≥ 14.0.0  
3. TypeScript ≥ 5.0.0
4. @shopify/cli latest
5. Polaris ≥ 12.0.0

!npm list @shopify/app

!npm list typescript
